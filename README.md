#Sudoku solver

This is a simple program reading a text file containing a sudoku.
the text file should look something like:

sudoku 1
003000245 <br>
010000600 <br>
600008000 <br>
400050000 <br>
200400600 <br>
700509103 <br>
030607020 <br>
000400000 <br>
500006000 <br>
sudoku 2
004007010
...


where a zero represent an empty square.
